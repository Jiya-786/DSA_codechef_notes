🧩 Problem Statement



You are given the head of a singly linked list.



Reverse the linked list and return the new head.



📌 You MUST do BOTH:

Iterative approach

Recursive approach



(Prof can ask either — or both)



🧱 Linked List Structure

struct ListNode {

&#x20;   int val;

&#x20;   ListNode\* next;

};

🔢 Input



Head of linked list:



Example:



1 -> 2 -> 3 -> 4 -> 5 -> NULL

📤 Output

5 -> 4 -> 3 -> 2 -> 1 -> NULL
