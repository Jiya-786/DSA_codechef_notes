🟢 Question 1: Valid Parentheses

🧩 Problem Statement



You are given a string s containing just the characters:



'(', ')', '{', '}', '\[' and ']'



Write a function to determine if the input string is valid.



✅ A string is valid if:

Open brackets must be closed by the same type

Open brackets must be closed in the correct order

Every closing bracket has a corresponding open bracket before it



🔢 Input

A string s (1 ≤ length ≤ 10⁴)

📤 Output

Return true if valid, otherwise false
